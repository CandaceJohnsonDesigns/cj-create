# WordPress Plugin Starter

This plugin was scaffolded with the `cj-create` CLI tool.

## Features

- Basic plugin header and safety checks
- Easy to extend into a full-featured plugin
- Includes GitHub Actions for PHP linting
- Includes GitHub Issue Templates

## Setup

1. Rename `plugin-name.php` to match your plugin slug.
2. Update headers and logic.
3. Build your plugin features.
